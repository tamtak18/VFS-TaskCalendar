## VFS TaskCalendar 公式ユーザーガイド

### 1．コンセプト

**VFS TaskCalendar** は，Vanilla JavaScriptのみで構築された超軽量・オフライン完結型の業務管理ツールです．一つのタスクデータを「カレンダー」「カンバン」「WBS」の3つの視点から可視化し，直感的な操作によるスケジュール管理を実現します．

### 2．基本操作

#### 2.1 タスクの作成

* **新規ボタン**: 画面右上の「+ NEW TASK（+ 新規タスク）」ボタンをクリックします．
* **ダブルクリック**: カレンダー上の特定の日付をダブルクリックすると，その日を開始・期限日とした状態で入力画面が開きます．

#### 2.2 タスクの編集・削除

* **編集**: カレンダーのバー，カンバンのカード，またはWBSのバーを直接クリックすると，プロパティ画面が開きます．
* **削除**: プロパティ画面内の「DELETE（削除）」ボタンをクリックします．

#### 2.3 スケジュールの調整（ドラッグ＆ドロップ）

* **カレンダー**: タスクバーを別の日付にドラッグすると，期間（工数）を維持したまま開始日・期限日を一括移動できます．
* **カンバン**: カードを「TODO」から「DONE」へドラッグすることで，進捗状態を更新できます．

### 3．画面解説

| タブ名称 | 用途 | 特徴 |
| --- | --- | --- |
| **CALENDAR** | 月・週・日単位の予定把握 | 表示切替が可能．土日は色分け表示されます． |
| **KANBAN** | 進捗ステータスの管理 | タスクを「未完了」「完了」の2つの列で整理します． |
| **WBS** | 中長期的な工数管理 | 今日から14日間のタイムラインを帯状に表示します． |

### 4．高度な機能

#### 4.1 アラートカラー・システム

タスクの緊急度をボーダー色で通知します．

* **緑**: 通常（TODO）
* **オレンジ**: 期限間近（デフォルトは期限3時間前）
* **赤**: 期限超過
* **グレー**: 完了済み（DONE）

#### 4.2 言語切り替え

ヘッダーの「JP/EN」セレクターにより，表示言語を即座に切り替え可能です．日本語モードではビジネス慣習に即した表記に最適化されます．

### 5．データ同期と保存

#### 5.1 自動保存

入力したデータはブラウザの `localStorage` に即座に保存されます．ブラウザを閉じてもデータが消えることはありません．

#### 5.2 端末間共有（OneDrive連携）

1. OneDrive上の任意の場所に `tasks.json` という名前の空ファイルを作成します．
2. 画面右上の「SYNC（同期）」をクリックし，そのファイルを選択します．
3. 別の端末でも同様にそのファイルを選択すれば，双方向での自動同期が開始されます（30秒周期）．
> **注意**: ブラウザのセキュリティ上，再起動時には再度「アクセス許可」や「同期設定」が必要になる場合があります．
